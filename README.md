# .unix

> Dotfiles for *nix systems.

- `bat` - `cat`
- `dust` - `du`
- `eza` - `ls`
- `htop` - `top`
- `fish` - Shell
- `niri` - Window manager
- `dunst` - Notification daemon
- `swaylock` - Screen locker
- `anyrun` - Application launcher
- `alacritty` - Terminal emulator
- `qutebrowser` - Web browser
- `neovim` - Text editor
- `pfetch` - System info
- `mpd` - Music server
- `ncmpcpp` - Music player
- `mpv` - Video player
- `cava` - Audio visualizer
- `zathura` - PDF viewer
